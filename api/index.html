    üöÄ API Documentation

    This service exposes the REST API layer for the Customer Loyalty & Analytics system. It powers the dashboard, segmentation visualizations, survival models, and automated email campaigns.

    Built with FastAPI, the API provides endpoints for querying customer data, analyzing transactions, generating RFM segments, and sending retention emails.

    ‚öôÔ∏è Tech Stack

    • FastAPI ‚Äì Lightweight, fast web framework for REST endpoints
    • SQLAlchemy ‚Äì ORM for interacting with PostgreSQL
    • Pydantic ‚Äì Request validation & API response schemas
    • Lifelines ‚Äì Kaplan-Meier survival analysis
    • BackgroundTasks ‚Äì Async email dispatching

    üë§ Customer Endpoints

    Manage customer profiles stored in DimCustomer:

    • POST /customers/ ‚Äì Create a new customer
    • GET /customers/{id} ‚Äì Retrieve customer details
    • PUT /customers/{id} ‚Äì Update customer profile
    • DELETE /customers/{id} ‚Äì Remove a customer record

    Also includes:

    • GET /analytics/customer-count-by-gender ‚Äì Gender-wise demographic distribution

    üí∞ Transaction & Revenue Analytics

    Explore sales data using joins over the star schema:

    • GET /revenue/monthly ‚Äì Monthly revenue totals
    • GET /customers/{id}/transactions ‚Äì Full transaction history of a customer
    • GET /analytics/transactions-by-store-month ‚Äì Monthly revenue by store
    • GET /analytics/transaction-amount-by-store ‚Äì Lifetime revenue by store

    üìä RFM Segmentation

    Provides data on Recency-Frequency-Monetary segments:

    • GET /analytics/customers-by-segment/{segment} ‚Äì List of customers in a specific segment
    • GET /analytics/segment-distribution/{all|male|female} ‚Äì Distribution of segments overall or by gender
    • GET /analytics/rfm-matrix ‚Äì RFM matrix for segment summaries

    üìß Email Campaigns

    Launch targeted retention campaigns via segment-specific email templates:

    • GET /analytics/segments_for_button ‚Äì List of all segments for frontend dropdowns
    • POST /campaigns/{segment} ‚Äì Asynchronously send emails to all users in a segment using BackgroundTasks

    Each template includes personalized subject lines and discount codes. See logic in EmailCampaignManager.

    üìà Survival Analysis

    Use Kaplan-Meier to visualize customer churn over time:

    • GET /analytics/survival-curve ‚Äì Returns survival probability at each time step using SurvivalData

    Supports visualization of customer retention lifecycle.
